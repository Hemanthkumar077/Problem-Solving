Check if a String is a Rotation of Another String 
Problem: Write a function to check if one string is a rotation of 
another string. 
Testcase 1: 
Input: "abcde", "cdeab" 
Output: true 

def is_rotation(s1: str, s2: str) -> bool:
    if len(s1) != len(s2):
        return False
    return s2 in (s1 + s1)

# Testcase
print(is_rotation("abcde", "cdeab"))  # True
print(is_rotation("abcde", "abced"))  # False
