Write a program that takes number of rows as input and print below 
respective pattern. 
Testcase1 :  Enter number of rows: 3 
Output      
:  
  A 
 A  B 
A  B  C 


code : 

n = int(input("enter a number: "))

cha = 64
for i in range(1,n+1):
    print(" " * (n-i), end="")
    for j in range(1,i+1):
        print(chr(cha + j), end=" ")
    print()
