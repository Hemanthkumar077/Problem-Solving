  Write a program that takes number of rows as input and print below 
respective pattern. 
Testcase1 :  Enter number of rows: 4 
Output      
:  
A  
AB 
ABC 
ABCD 


code : 

n = int(input("enter a number: "))
ch = 64
for i in range(1,n+1):
    for j in range(1,i+1):
        digit = ch + j
        print(chr(digit),end="")
    print()
